===========================================
Splinter - Курсовой проект
===========================================

Разработчик: Kira
Год: 2026
Движок: Unity 2022+
Язык: C#

===========================================
ОПИСАНИЕ
===========================================
Шутер от первого лица с системой сохранений,
базой данных пользователей и RPG-элементами.

===========================================
УПРАВЛЕНИЕ
===========================================
WASD - Движение
Shift - Бег
ЛКМ - Стрельба
ПКМ - Удар
Shift+Прокрут колесика мыши - Смена оружия
C - Перезарядка
Esc - Меню / Пауза
F1 - Помощь
F2 - Об игре
Ctrl+S - Сохранение

===========================================
ТРЕБОВАНИЯ
===========================================
- Windows 10/11
- .NET Framework 4.8+
- Видеокарта с поддержкой DirectX 11

ЗАПУСК
===========================================
